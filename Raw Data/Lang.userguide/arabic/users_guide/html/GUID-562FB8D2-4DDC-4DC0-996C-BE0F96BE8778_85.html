تغيير الوضع الافتراضي

الطرز ذات العلاقة:DCP‑L2520D / DCP‑L2540DW / DCP‑L2541DW / MFC‑L2700D / MFC‑L2701D / MFC‑L2700DW / MFC‑L2701DW
عندما يكون الجهاز خاملاً أو عندما تضغط على Stop/Exit ‏(توقف/خروج)‏، سيعود الجهاز إلى الوضع الذي قمت بتعيينه.
  • بالنسبة لطرز MFC، يتضمن الجهاز أوضاع الفاكس والمسح الضوئي والنسخ. إعداد المصنع هو وضع الفاكس.
  • بالنسبة لطرز DCP، يتضمن الجهاز وضعي ‏«المسح الضوئي»‏ و‏«النسخ»‏. يتم تثبيت وضع ‏«النسخ»‏ كوضع افتراضي.
  1. اضغط على Menu ‏(القائمة)‏.
  2. اضغط على أو لعرض الخيار General Setup ‏(الإعداد العام)‏، ثم اضغط على OK ‏(موافق)‏.
  3. اضغط على أو لعرض الخيار Default Mode ‏(الوضع الافتراضي)‏، ثم اضغط على OK ‏(موافق)‏.
  4. اضغط على أو لعرض Fax ‏(الفاكس)‏ أو Copy ‏(نسخ)‏ أو الخيار Scan ‏(مسح ضوئي)‏، ثم اضغط على OK ‏(موافق)‏.
  5. اضغط على Stop/Exit ‏(توقف/خروج)‏.
