بروتوكولات ‏(IPv4)‏

ARP وRARP وBOOTP وDHCP وAPIPA‎(Auto IP)‎ ودقة اسم WINS/NetBIOS وDNS resolver وmDNS ومستجيب LLMNR وLPR/LPD ومنفذ خام خاص/Port 9100 وIPP وخادم FTP وSNMPv1/v2c/v3 وخادم HTTP وخادم وعميل TFTP وعميل SMTP وICMP وWeb Services ‎(Print/Scan)‎ وعميل SNTP ‏(لطرز MFC)‏، حساب POP3 ‏(عند تنزيل الفاكس عبر الإنترنت.)‏ ، IMAP4 ‏(عند تنزيل الفاكس عبر الإنترنت.)‏

بروتوكولات ‏(IPv6)‏

NDP وRA وDNS Resolver وmDNS ومستجيب LLMNR وLPR/LPD ومنفذ خام مخصص Port 9100 وIPP وخادم FTP وSNMPv1/v2c/v3 وHTTP Server وعميل وخادم TFTP، وعميل SMTP وICMPv6 و Web Services ‎(Print/Scan)‎ وعميل SNTP ‏(لطرز MFC)‏ حساب POP3 ‏(عند تنزيل الفاكس عبر الإنترنت.)‏ ، IMAP4 ‏(عند تنزيل الفاكس عبر الإنترنت.)‏

أمان الشبكة ‏(السلكية)‏
